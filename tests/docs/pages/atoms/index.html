Atoms Overview

Atoms are the foundational building blocks of the KitCLA design system. These are the smallest possible components that can't be broken down any further - buttons, inputs, icons, and other basic HTML elements with specific styling and behavior.

Buttons

Interactive elements for user actions including primary, secondary, and Alpine.js variants.

Inputs

Form input elements including text, number, select, file upload, and specialized inputs.

Cells

Data display components for tables and grids including text, pills, booleans, and time cells.

Icons

SVG icon components with consistent sizing and styling options.

Headers

Typography components for titles, headings, and section headers.

Links

Clickable text and navigation elements with hover states and styling.
